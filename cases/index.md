# Casos de Threat Hunting - Escenarios Completos

Simulaciones de ataques completos (kill chain, NIST, Pirámide del dolor) con análisis paso a paso, desde detección inicial hasta respuesta.


## Diferencia entre Técnicas y Casos

| Técnicas | Casos |
|----------|-------|
| Análisis de **una sola técnica MITRE** | Análisis de **múltiples técnicas** encadenadas |
| Enfoque en detección específica | Enfoque en investigación completa |
| Metodología estructurada y repetible | Escenario de ataque realista |
| Output: Regla de detección | Output: Reporte de incidente |

## Estructura de Cada Caso

Todos los casos siguen este template:
```
1. Executive Summary
2. Contexto del Escenario
3. Kill Chain del Ataque
4. Setup del Laboratorio
5. Timeline de Eventos
6. Análisis Técnico Detallado
7. IoCs Extraídos
8. Reglas de Detección Creadas
9. Playbook de Respuesta
10. Lecciones Aprendidas
11. Referencias
```

## Habilidades Desarrolladas

Al completar estos casos, desarrollarás:

**Análisis forense de incidentes**  
**Correlación de eventos multi-fuente**  
**Desarrollo de detecciones personalizadas**  
**Interpretación de kill chains**  
**Documentación técnica profesional**  
**Respuesta a incidentes (IR)**
